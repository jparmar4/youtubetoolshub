# Q3 2026 Organic Traffic & Revenue Content Plan

## Decision summary

Publish **two new, source-backed articles per week** and improve **one existing URL per week**. This replaces the previous 36-new-post calendar, which would overlap heavily with the current catalog and risk keyword cannibalization.

Do not choose subjects only because their advertiser CPC is high. Search intent, genuine expertise, a useful related tool, and an achievable ranking opportunity matter more at the site's current traffic level.

## How to use the numbers

| Metric | Meaning | Use it correctly |
|---|---|---|
| Global demand | Directional average monthly Google search-demand range for the English query family | Validate in Google Ads Keyword Planner before writing. It is not a guaranteed traffic number. |
| Six-month traffic target | Monthly organic visits per page after six months, assuming useful content and a new/small domain | Track against Google Search Console; revise after 8–12 weeks. |
| PPC CPC | Planning range for advertiser cost per click, mainly US/UK/CA/AU commercial intent | This is **not** your AdSense earnings per click. |
| Page CPM / RPM | Planning range per 1,000 page views when a meaningful share of visitors is Tier 1 | Actual value depends on country, season, ad fill, viewability, and consent. |

All financial fields below are directional USD planning ranges, not observed Keyword Planner or AdSense values. Replace them with account data before using them in a forecast.

## Measurement rules

1. In Keyword Planner, set language to English and compare Global with United States, United Kingdom, Canada, and Australia.
2. In Search Console, map one primary keyword to one canonical URL. If an existing URL already receives impressions for the term, refresh that URL instead of publishing another article.
3. Review each page after 8 weeks: impressions, CTR, average position, organic visits, engaged sessions, page RPM, and outbound/tool clicks.
4. Do not claim exact RPM, payment, legal, tax, insurance, or sponsorship figures without a dated primary source.

## Revenue target model

Use this only after Search Console gives real traffic data:

`monthly page revenue = organic page views ÷ 1,000 × actual page RPM`

Example: 5,000 monthly organic page views at a $3.50 actual page RPM = **$17.50/month**. Increasing qualified Tier 1 traffic and engagement improves the opportunity; it does not guarantee an RPM.

## Per-post definition of done

- One primary keyword and one canonical URL.
- A source-backed answer in the first 100 words.
- Original examples, screenshots, comparison, checklist, or data—not generic AI text.
- One directly relevant tool CTA and two contextual internal links.
- Author/reviewer and a visible update date for factual pages.
- Search Console indexing request after publishing; no artificial traffic, clicks, or link spam.
